Mac heavy lovers must see how to efficiently maintain your Mac

2017-07-30
Efficiently, Mac, Maintain

Like all computers, Mac is a long time and it’s prone to a system crash. Imagine your design draft done half, or is going to run the code, suddenly, your computer relentless shut down! The The Wow, really can not think of life can have any worse things, and probably the rooftop met it.

Mac after the period with, due to physical components (logic board, RAM, internal fan, etc.) failure, it is likely to lead to computer operating efficiency, running is not regular. At this time, your computer will be unable to open the file, search speed and accuracy decline, the application of abnormal circumstances.

However, everything is beyond the way of the road drops, the following we will give you some maintenance programs to help you reduce the possibility of problems.

  • Clever use of Apple Diagnostics to find hardware problems
  • Monitor and maintain the health of the hard disk

Use Apple Diagnostics to find hardware problems

Mac’s hardware components are fairly robust. But over time, the failure may still occur, although the Mac may continue to run, but some failures can lead to hardware or software instability.

Apple Diagnostics or earlier Apple Hardware Test is a comprehensive tool for determining hardware problems within Mac. This includes logic boards, memory, batteries or power adapters, sensors, graphics, and wireless components. It can be tested to ensure that all components and sensors are working properly.

There are many ways to run Apple Diagnostics or Apple Hardware Test. Although their appearance and name are different, but they can test the hardware problem. This depends largely on its model and year.

  • Since June 2013: Macs uses the new hardware test system Apple Diagnostics. It is a hidden disk image that can also run on the internet
  • From mid-2011 to June 2013: Apple discontinued drive. Mac runs Apple Hardware Test in Recovery HD. If not available, they can run on the Internet
  • By mid-2011: Mac with CD-ROM and OS X installation CD can not be made.

Apple Diagnostics test program

Step 1. If you have an Apple hardware test tool on a DVD or USB drive, insert it into your Mac. If the Mac does not contain external boot media, make sure the Mac is connected to the Internet.

Step 2. Disconnect all external devices except the keyboard, mouse, monitor, Ethernet cable, and AC power. If you are testing on a laptop, make sure to connect to AC power. Also make sure the mouse and keyboard are connected to the computer via wired USB instead of Bluetooth.

Step 3. Make sure the Mac is on a hard, flat, stable surface and well ventilated.

Step 4. Select Apple menu> Shutdown and wait for Mac to turn off completely.

Step 5. Open Mac and then press and hold the D key on the keyboard immediately. Continue to hold the key until the Mac’s gray screen changes to a hardware test or Apple Diagnostics.

Step 6. Hold down the Option-D key to force the system to load the tool from the Internet.

Continue to hold the key until you see the “Start Internet Recovery” message on your Mac. Select the network from the drop-down menu and wait for the language interface to appear.

If the hardware test or Apple Diagnostics starts successfully from the USB drive or Recovery HD, skip this step.

Step 7. Select the language, then click the right arrow or press the Back key.

Step 8. Click the Hardware Test tab. Apple hardware testing supports two types of tests - standard and extended testing.

Click OK in the Perform Extension Test box, and then click the Test button. The latter test takes more time to finish.

If the test goes smoothly, the test area will show no problems or pass Pass on all applicable tests.

If the test is done smoothly, no problems are found or the issues are listed and the details are displayed.

Step 9. During the testing process, you may hear the fan buzzing up and down, but this is part of the process. In the latest Mac, you will see a gray progress bar that shows Checking your Mac … roughly the rest of the time.

Finally, click Restart to restart Mac.

What if the hardware test can not be started?

It is recommended that you turn off the computer, and then hold down the D key until a gray screen appears. If the gray screen fails, wait for the Mac to start, and then close and repeat the process.

If you reinstall the old computer from scratch and unfortunately lose the original disk that came with the hardware test tool, the diagnostic tool may no longer be available. But you can save! You can also restore hardware testing tools. Use the following terminal commands to determine the model and board number of the Mac

You need to jump to Apple Hardware Test Github Page and download your Mac related packages. Copy the package to the /System/Library/CoreServices/.diagnostics folder and then restart it while holding down the D to start the diagnostic tool.

Explain the results

When a test encounters any errors, a failure message appears as well as a reference code and a brief description. E.g

  • NDK001,003,004 - Keyboard problems
  • NDC001, 003 … to 006 - camera release
  • NDR001,003,004 - with the touchpad
  • PFM001, 0000 … to 007 - issued with SMC
  • NDL001 problem with Bluetooth hardware

If you encounter any hardware errors, you must move the Mac to the nearest Apple Store and replace it. Before you call Apple for support, please remember the error code and warranty card.

You should run Apple Diagnostics Tests when the following conditions occur:

  • Whenever you install new RAM or any new hardware components
  • When you start to appear that can not resolve the disk utility or the problem of reinstalling the operating system
  • When your Mac is used for more than three years longer

Alternatives to the Apple Diagnostics tool

There are many third-party utilities that can perform specific types of hardware tests and provide you with more information.

  • Memtest, found intermittent memory problems, which is a common cause of collapse or freezing that can not be explained in the Mac. Users running Mac OS X 10.6.8 or earlier must download version 4.22. For 64-bit Macs download version 4.23
  • Techtool Pro, A comprehensive application for checking for file system errors, I / O errors, POST errors, internal battery conditions, hard drive health (SMART), Bluetooth hardware, and network adapters. The cost of the application is $ 99, but it can provide far more details than the Apple Diagnostics
  • Atomic, Dedicated RAM checker that can detect memory problems by using various memory test techniques. The cost of this application is $ 29.99, which can be detected in advance

Monitor and maintain the health of the hard disk

As time goes on, almost all of the disks are likely to have smaller bugs. Since these errors may occur during normal computer use, the CD must be regularly monitored and maintained. Frequent errors can damage the file system, resulting in slowing down, crashing, or losing data.

Fixing hard drives is usually more troublesome than troubleshooting steps; however, this is considered a precaution, taking into account the frequency of random discs, hard disk failure rates, and occasional irregularities.

Monitor the health of the hard drive

Most modern hard drives and SSDs have built-in sensors and supervisory circuits that form a system called SMART that means self-monitoring, analysis and reporting techniques.

The purpose of this technology is to provide a report on the various specifications of the reliability of the drive, with the aim of finding an imminent hardware failure before it occurs.

To check the SMART status of the drive, open the Disk tool, in Applications> Tools, and then select the drive in the list on the left. If the selected drive supports SMART, you will see this message at the bottom of the window: S.M.A.R.T. Status: Verified.

If you see the red letters about the failure, please back up the drive immediately, although it is hard to say how much data you will be able to back up. Even if the disk utility can solve the problem, you should also back up the data and replace the disk as soon as possible.

If the Mac has a fused drive, or if FileVault is enabled, the disk utility may not show SMART status even if your hardware supports SMART. Open the system information, select the store in the list on the left, and then locate the SMART status on the right.

Although the disk utility reports SMART status, but in fact you do not get an uninterrupted real-time drive reliability and its parameter reporting. There are many third-party utilities that monitor SMART status in detail and remind you of any changes.

Smartmontools is a command-line utility consisting of two packages (s martctl and smartd). They use SMART to control and monitor the storage system and provide you with information on ongoing disk degradation through various statistics.

Backblaze uses these reliability statistics to predict hard drive failures.

  • SMART 5 - Redistribution industry number
  • SMART 173- wear balance count
  • SMART 187 - Reported error that could not be corrected
  • SMART 197 - Number of sectors currently pending
  • SMART 198 - Number of non-correctable sectors

Download and install the latest package. Open the terminal and enter diskutil list. This command will populate all disks (either internal or external) with its size and identifier id.

execute：

This command generates a long report of the drive. If the RAW value for any of the statistics is greater than zero, no investigation is required, although you may also see a list of values in percent. Any less than 60% of the need for your attention.

You can also try these third-party utilities to provide you with more features and a good user interface.

  • SMARTReporter is a SMART-based utility that also checks for I / O errors or RAID (Redundant Array of Inexpensive Disks). It is priced at $ 5.99 on the Mac App Store
  • SMARTUtility - This utility is also based on SMART. The SMART Utility not only displays individual attributes to view its status and information, but also uses an internal algorithm based on these attributes to detect a drive that failed before the SMART indication failed. Price $ 25
  • DriveDX - It provides a complete, extensive drive inspection and monitoring tool for Mac, the price is $ 20. It has an intuitive explanatory UI that can help you understand the meaning of each property. It also uses the drive health metrics that are most closely related to the drive failure to perform the overall drive health diagnostics

Keep the hard drive healthy

In addition to regular monitoring, you also need to maintain the hard disk. There are many examples here

  • After a power outage, the hard reboot and the system crash - especially the Mac exception
  • Periodically, as part of a routine maintenance routine, identify any hidden directory corruption before becoming a problem
  • Software-related errors may cause you to write corrupted data on your disk or worse, lost data

Open the Disk Utility app. Select Start from the sidebar and click the “First Aid” button on the toolbar. When you click Run, the Disk Utility will verify the disc and try to repair it if necessary.

If the Disk Utility finds any problems that need to be fixed, you will need to start your Mac from Recovery or Internet Recovery. Wait for the utility window to open, and then click the Disk tool.Select the boot disk, and then click First aid.

In rare cases, the Disk Utility encounters a serious problem that can not be solved and you may need to use a third-party utility:

  • DiskWarrior - a utility for preventing and resolving directory corruption, which is the main reason for Mac instability. DiskWarrior rebuilds the disk directory by using the data recovered from the original directory to resolve disk corruption, thereby recovering files and folders that you think are missing. Price $ 120.
  • Techtool Pro - A comprehensive application for checking for file system errors, I / O errors, POST errors, internal battery conditions, hard drive health (SMART), Bluetooth hardware, and network adapters. The app is priced at $ 99, but can provide you with details that go far beyond Apple Diagnostics.

Conclusion

In this part of this tutorial, we show how to maintain Macs by periodically performing Apple Diagnostics to check and hard disk authentication.

Hard drives or solid-state drives are an important part of the Mac and OS operating systems. Although each hard drive will eventually be scrapped in a few years, you can keep your information and protect your data through regular backups and CD maintenance.
